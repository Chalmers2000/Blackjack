
# SPECIFICATION

## GOALS
1. This is a Blackjack game that can be played standalone on a Windows 11 desktop.
2. The game is built using HTML, CSS, and JavaScript: Store the appropriate elements of the game in the appropriate files.
3. The game should closely emulate the look and feel of the baccarat game baccarat.html, you can see the user interface at baccarat-image.png. Closely imitate the look of the chips
4. The game should closely emulate the animation and timing of the baccarat.html game.

## REQUIREMENTS
1. The game supports blackjack play, including hit, stand, split, and double-down.
2. The game keeps track of the win/loss record and earnings in a table across the bottom, just like the baccarat game did.
3. The player's odds of winning the hand are displayed in the corner, and re-calculated after each card is dealt.
4. There is a six-shoe deck.
